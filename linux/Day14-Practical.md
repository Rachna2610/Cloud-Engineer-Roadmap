# Day 14 Practical – Linux Networking Commands

## Objective

Practice basic Linux networking commands used for troubleshooting.

## Commands Practiced

### Display IP address

```bash
ip addr
```

### Display network interfaces

```bash
ip link
```

### Display routing table

```bash
ip route
```

### Display hostname

```bash
hostname
```

### Display DNS configuration

```bash
cat /etc/resolv.conf
```

### Test internet connectivity

```bash
ping -c 4 google.com
```

## Outcome

- Displayed IP address.
- Viewed network interfaces.
- Checked routing table.
- Displayed system hostname.
- Viewed DNS configuration.
- Verified internet connectivity.
