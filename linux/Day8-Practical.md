# Day 8 Practical - Shell Scripting Basics

## Objective

Learn how to:

- Create shell scripts
- Execute shell scripts
- Use variables
- Take user input
- Use Linux commands inside scripts

---

## Commands Used

```bash
chmod +x hello.sh
./hello.sh
./variable.sh
./input.sh
./date.sh
./info.sh
```

---

## Outcome

Successfully:

- Created shell scripts
- Executed scripts using Bash
- Used variables and user input
- Displayed system information
