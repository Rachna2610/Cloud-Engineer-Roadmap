# Linux Introduction

## What is Linux?

Linux is a free and open-source operating system developed by Linus Torvalds in 1991. It acts as an interface between computer hardware and software, allowing users and applications to communicate with the system.

Linux is widely used in servers, cloud computing, mobile devices, embedded systems, and supercomputers because of its stability, security, and performance.

---

## History of Linux

- In 1991, Linus Torvalds created the Linux kernel as a personal project.
- The Linux kernel was released under the GNU General Public License (GPL).
- Developers around the world contributed to its growth.
- Today, Linux powers a large portion of the internet, cloud platforms, and enterprise systems.

---

## Features of Linux

### 1. Open Source
The source code is freely available and can be modified by anyone.

### 2. Multi-user
Multiple users can access the system simultaneously.

### 3. Multitasking
Linux can run multiple applications at the same time.

### 4. Security
Linux provides strong security through user permissions and authentication.

### 5. Portability
Linux can run on different hardware platforms.

### 6. Stability
Linux systems can run for long periods without crashing.

### 7. Command Line Interface (CLI)
Linux provides powerful command-line tools for system administration.

- Reliable
- Cost-effective
