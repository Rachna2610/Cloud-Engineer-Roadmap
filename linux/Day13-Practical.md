# Day 13 Practical – Archive & Compression

## Objective

Learn how to create, compress, and extract archive files in Linux.

## Commands Practiced

### Create a TAR archive

```bash
tar -cvf project.tar project
```

### Create a compressed TAR archive

```bash
tar -czvf project.tar.gz project
```

### Extract a TAR archive

```bash
tar -xvf project.tar -C extract_tar
```

### Extract a compressed TAR archive

```bash
tar -xzvf project.tar.gz -C extract_gz
```

## Outcome

- Created a TAR archive.
- Created a compressed TAR archive.
- Extracted archive files.
- Understood how Linux administrators and Cloud Engineers create backups.
