# Day 12 Practical – User & Group Management

## Objective

Practice Linux commands for viewing users, groups, and account information.

## Commands Practiced

### Display current user

```bash
whoami
```

### Display user information

```bash
id
```

### Show logged-in users

```bash
who
```

### Display user groups

```bash
groups
```

### View user account information

```bash
cat /etc/passwd | head
```

## Outcome

- Identified the current user.
- Viewed user ID and group information.
- Checked logged-in users.
- Displayed group memberships.
- Viewed system user accounts.
