# Day 6 - Services and systemctl Practical

## Commands Practiced

```bash
systemctl list-unit-files --type=service
systemctl status cron
systemctl is-enabled cron
sudo systemctl restart cron
```

## Learning Outcome

- Listed installed services
- Checked service status
- Verified enabled services
- Restarted a Linux service
- Understood service management using systemctl
- Learned service administration used on AWS EC2 servers
