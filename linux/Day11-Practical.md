# Day 11 Practical – Disk & Storage Management

## Objective

Practice Linux disk and storage management commands.

## Commands Practiced

```bash
df -h
```

Displays disk usage in a human-readable format.

```bash
du -sh .
```

Displays the size of the current directory.

```bash
lsblk
```

Lists available storage devices and partitions.

```bash
mount | head
```

Displays the first few mounted file systems.

```bash
free -h
```

Displays system memory usage.

## Outcome

- Checked disk usage.
- Checked directory size.
- Viewed storage devices.
- Viewed mounted file systems.
- Monitored memory usage.
