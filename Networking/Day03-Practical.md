# Day 03 Practical - TCP/IP Model

## Objective

Understand TCP and UDP communication using Linux networking commands.

---

## Commands Practiced

### View TCP and UDP Listening Ports

```bash
ss -tuln
```

Displays all listening TCP and UDP ports.

---

### Display TCP Connections

```bash
ss -t
```

Shows active TCP connections.

---

### Display UDP Connections

```bash
ss -u
```

Shows UDP sockets.

---

### Display TCP and UDP Network Connections

```bash
ss -tun
```

Displays TCP and UDP network connections in numeric format.

---

## What You Learned

- Viewed active TCP connections.
- Viewed UDP sockets.
- Observed listening network services.
- Understood the Transport Layer of the TCP/IP Model.
- Compared TCP and UDP using real Linux commands.

Successfully explored TCP and UDP communication using the `ss` command and related the results to the Transport Layer of the TCP/IP Model.
